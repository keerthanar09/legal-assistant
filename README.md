## Getting Started

### Prerequisites
- Node.js (v16+)
- npm

### Installation
```bash
git clone https://github.com/yourusername/legal-assistant.git
cd legal-assistant
npm install
```

### Environment Variables

### Running the App
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.
